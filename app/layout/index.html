What Is Medal

Medal is a static blog generator written in Node and ES2015, utilising Markdown, Nunjucks and HighlightJS to build a lightening fast static site that can be hosted anywhere.

Progressively Enhanced

Medal is designed to produce fast static HTML pages which will work in the browser with or without JavaScript. Luxuries such as filtering and search are progressively added.

Open For Extension

Client facing JavaScript can be extended simply by adding a new component and associating it to a DOM element with a jQuery selector.
