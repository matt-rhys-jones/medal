What Is Medal

Medal is a static blog generator written in Node and ES2015, utilising Markdown, Nunjucks and HighlightJS to build a lightening fast static site that can be hosted anywhere.

Simplicity Over Abstraction

There are many static site generators out there, most of which are genuinely brilliant, but have been heavily abstracted to allow for swappable template engines, themes and features. Medal uses the tools you (probably) already know in order to deliver an easy to maintain, yet extendible, static site generator.
